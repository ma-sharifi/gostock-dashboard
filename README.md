# GoStock Dashboard

A stock portfolio dashboard built with React, Tailwind CSS, and Recharts.

## Getting Started

Install dependencies:
```bash
npm install
```

### Development (Vite)
```bash
npm run dev
```

### Build
```bash
npm run build
```

### Serve with @web/dev-server
Build and serve in one command:
```bash
npm run serve
```

Or serve an existing build:
```bash
npm start
```

The app will open at **http://localhost:8000**.

## Tech Stack

- **React 18** — UI framework
- **Tailwind CSS 3** — Utility-first styling
- **Recharts** — Chart library
- **Lucide React** — Icon set
- **Vite** — Build tool
- **@web/dev-server** — Modern web dev server (serves production build)
